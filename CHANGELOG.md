# Changelog

Todos as mudanças notáveis deste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [2.0.0] - 2025-01-15

### Adicionado
- 🎉 Refatoração completa com arquitetura modular
- 📦 Suporte a instalação via pip com pyproject.toml
- 🐳 Suporte a Docker e Docker Compose
- 🔄 CI/CD com GitHub Actions
- 🧪 Testes abrangentes com pytest
- 📝 Sistema de logging configurável
- ⚙️ Sistema de configuração avançado com variáveis de ambiente
- 📢 Sistema de notificações melhorado via Telegram
- 💾 Sistema de banco de dados com SQLite
- 📊 Sistema de geração de gráficos com Matplotlib
- 🎯 Interface de linha de comando completa
- 🔒 Suporte a proxy e rate limiting
- 💾 Sistema de backup automático do banco de dados
- 📚 Documentação detalhada e exemplos de uso
- 📄 README melhorado com badges e instruções detalhadas
- 🤝 Guia de contribuição (CONTRIBUTING.md)
- 📄 Licença MIT

### Alterado
- 🏗️ Estrutura de diretórios reorganizada
- 💻 Código refatorado para melhor manutenibilidade
- 📝 Documentação atualizada e expandida
- 🔧 Dependências atualizadas
- 🎨 Seletores CSS melhorados
- ⚡ Performance otimizada

### Removido
- ❌ Código legado do Colab
- ❌ Arquivos duplicados

### Corrigido
- 🐛 Tratamento de erros melhorado
- 🔧 Correções de bugs na extração de dados
- 💾 Correções no salvamento de arquivos
- 📢 Correções no sistema de notificações
